# String#to_babi
## Installation

Add this line to your application's Gemfile:

```ruby
gem 'to_babi'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install to_babi

## Usage

    require 'to_babi'

    'こんにちは'.to_babi #=> 'こぼんぶにびちびはば'
